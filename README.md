# AI投标助手V1.0

## 系统简介

本系统旨在帮助企业高效管理投标业务，通过智能化手段提升投标文件编制效率和质量。系统整合企业知识库、招标项目管理、投标文件生成与检查等核心功能，实现从招标信息获取到投标文件提交的全流程数字化管理。

### 核心价值

- ✅ **提高投标效率** - 自动化文件生成，节省大量人工编制时间
- ✅ **降低废标风险** - 智能检查功能，确保投标文件完整合规
- ✅ **统一知识管理** - 集中管理企业资质、业绩、人员等核心资料

---

## 功能模块

### 1. 企业知识库

集中管理企业基本信息、资质证书、财务数据、业绩案例、人员信息等核心资料。

**主要功能：**
- 📋 公司基础信息管理
- 🏆 资质信息管理
- 💰 财务信息管理
- 📊 业绩信息管理
- 👥 人员信息管理
- 📁 历史文档管理

**使用场景：**
- 录入和维护企业基本信息、统一社会信用代码等
- 上传和管理各类资质证书（营业执照、资质证书等）
- 记录财务报表数据（年度、季度财务信息）
- 维护历史业绩案例（项目名称、金额、合同等）
- 管理企业人员信息（学历、资质、证书等）

---

### 2. 招标项目管理

上传招标文件，自动解析目录结构，提取关键信息，实现招标项目的统一管理。

**主要功能：**
- 📤 文件上传解析
- 📑 目录结构识别
- 📝 项目信息管理
- 🔄 状态跟踪

**使用场景：**
- 上传招标文件（Word、PDF等格式）
- 自动解析招标文件目录结构
- 提取招标项目关键信息（项目名称、预算、要求等）
- 跟踪项目进度和状态

---

### 3. 投标文件生成

基于招标要求和企业知识库，智能生成投标文件，支持自定义格式和内容。

**主要功能：**
- 🎯 智能选择项目
- 🔗 自动匹配资料
- 📏 字数控制
- 📄 Word导出

**使用场景：**
- 选择招标项目
- 自动匹配企业知识库中的相关资料
- 根据招标要求生成对应章节内容
- 控制字数、格式等参数
- 导出标准Word投标文件

---

### 4. 投标文件检查

自动检查投标文件完整性、合规性，降低废标风险，提供详细的检查报告。

**主要功能：**
- ✓ 完整性检查
- ✓ 合规性验证
- 🎯 问题定位
- 📋 检查报告

**使用场景：**
- 上传待检查的投标文件
- 自动检查文件完整性（是否缺页、缺章节等）
- 验证文件合规性（格式、签章、密封等要求）
- 生成详细的检查报告
- 定位问题并提供修改建议

---

### 5. 系统管理

用户、角色、权限、企业的统一管理，支持多企业、多角色的权限控制。

**主要功能：**
- 👤 用户管理
- 🛡️ 角色管理
- 🔐 权限控制
- 🏢 企业管理
- 📊 数据权限

**使用场景：**
- 创建和管理系统用户
- 配置用户角色和权限
- 管理多个企业账号
- 设置数据访问权限
- 审计用户操作日志

## 使用指南

### 快速开始

1. **首次使用建议**
   - 使用 `admin` 账号登录，熟悉系统整体功能
   - 在"企业管理"中创建企业账号
   - 在"用户管理"中创建企业用户并分配角色

2. **日常业务操作**
   - 使用 `company_admin` 账号登录
   - 在"企业知识库"中维护企业资料
   - 在"招标项目管理"中上传招标文件
   - 在"投标文件生成"中生成投标文件
   - 在"投标文件检查"中检查投标文件

3. **基础查看操作**
   - 使用 `user` 账号登录
   - 查看招标项目列表
   - 执行简单的文件生成操作

---

### 业务流程

#### 完整投标流程

```
1. 获取招标信息
   ↓
2. 上传招标文件（招标项目管理）
   ↓
3. 系统自动解析目录结构
   ↓
4. 维护企业知识库（如需更新）
   ↓
5. 选择项目生成投标文件（投标文件生成）
   ↓
6. 系统自动匹配资料并生成
   ↓
7. 检查投标文件（投标文件检查）
   ↓
8. 修改完善后导出Word文件
   ↓
9. 打印装订提交
```

#### 知识库维护流程

```
1. 登录企业管理员账号
   ↓
2. 进入"企业知识库"模块
   ↓
3. 选择对应子模块：
   - 公司信息
   - 资质信息
   - 财务信息
   - 业绩信息
   - 人员信息
   ↓
4. 新增或编辑资料
   ↓
5. 上传相关附件
   ↓
6. 保存数据
```

## 常见问题

### Q1: 如何重置密码？

**A:** 当前版本暂不支持密码重置，请联系系统管理员重置账号。

### Q2: 上传的文件存储在哪里？

**A:** 开发环境中，文件信息存储在LocalStorage中（实际文件未上传）。生产环境中，文件会上传到Supabase Storage。

### Q3: 如何导出投标文件？

**A:** 在"投标文件生成"模块中，完成文件生成后，点击"导出Word"按钮即可下载。

### Q4: 系统支持多少个企业？

**A:** 系统支持多企业管理，理论上没有数量限制。每个企业的数据相互隔离。

### Q5: 如何配置数据权限？

**A:** 在"系统管理 > 角色管理"中配置角色权限，在"企业管理"中设置数据权限范围。
